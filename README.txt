Recebi um trabalho para realizar conversões de PDFs em arquivos de imagens, era um trabalho ardo, e demorado, cansativo  e totalmente viavel para ser feito de forma
automaizada, ai veio a ideia de criar um script que realizasse as conversões de forma automatizada.
*****
Para utilizar, basta colocar o executavel(ou script caso tenha o python instalado) dentro da pasta onde contenha o pdf com as diversas paginas), que o mesmo se encarrega de 
analisar quantas paginas tem, e fazer a conversão das mesmas, em aquivos no formato de imagem.jpg.
